How to Set Up Your API Key:

  1. Sign up for a free API key at GNews.io or NewsAPI.org
  2. Open the script.js file in a text editor
  3. Find the line: const API_KEY = 'YOUR_API_KEY';
  4. Replace 'YOUR_API_KEY' with your actual API key
  5. Save the file and refresh the page
